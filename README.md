# NFC Tab Bomb 50+ для Flipper Zero

## 📱 Что это?
NFC карта, которая при поднесении к телефону открывает 50+ вкладок в браузере.

## 🎯 Эффект
- **Android**: 50+ вкладок открываются автоматически
- **iOS**: Требует подтверждения, но всё равно бесит
- **Desktop**: Тоже работает (для тестов)

## 🔧 Установка на Flipper Zero

1. Скачай файл `nfc/tab_bomb.nfc`
2. Подключи Flipper к компьютеру
3. Скопируй файл в папку `/nfc` на Flipper
4. Открой NFC → Read → Load from file
5. Выбери `tab_bomb.nfc`
6. Нажми "Emulate"
7. Поднеси к телефону жертвы!

## ⚠️ Предупреждение
Только для образовательных целей! Не используй для реального вреда.
